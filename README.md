# 🍽️ Sistema de Restaurante

## 📌 Descrição
Sistema desenvolvido para gerenciamento de pedidos em restaurante, auxiliando no controle de atendimento e organização das informações.

O sistema permite registrar pedidos e facilitar o fluxo de atendimento.

## 🚀 Tecnologias
- C#
- ASP.NET Core
- MariaDB (banco de dados relacional compatível com MySQL)
- HTML, CSS, JavaScript, Bootstrap

## 🎯 Funcionalidades
- Cadastro de pedidos
- Controle de clientes
- Organização dos pedidos
- Listagem de registros
- Cálculo por pedido

## 📚 Aprendizados
- Estruturação de sistema web
- Organização de dados
- Manipulação de banco relacional
- Implementação de funcionalidades CRUD

## 📌 Melhorias futuras
- Integração com sistema de pagamento
- Melhorias na interface
